# HTML 游戏合集

经典游戏的HTML/JavaScript实现合集。

## 游戏列表

### 1. 坦克大战 (`tank-battle.html`)
经典坦克大战游戏。
- **操作**: ↑↓←→ 移动，空格射击，P暂停
- 随机地图、砖墙/钢墙/水域地形、AI敌人、多关卡

### 2. 超级马里奥 第一关 (`mario-level1.html`)
超级马里奥1-1风格平台跳跃游戏。
- **操作**: ←→ 移动，空格/↑ 跳跃，P暂停
- 210列宽幅关卡、管道、问号块、板栗仔敌人、旗杆通关

## 如何游玩

直接双击对应的 `.html` 文件即可在浏览器中游玩。
